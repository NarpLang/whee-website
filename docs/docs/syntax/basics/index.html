Basic Syntax

Syntax Information

All Syntax is stored in a simple file: /etc/whee/syntax.json

You can modify it at ay time!

Let's look at the print_string object:

"print_string": { "type": "function", "pattern": "print\\(\"([^\"]*)\"\\);", "replacement": "printf(\"%s\\n\");" }

Type: function (This tells The WheeC Engine that it's a function)

Pattern: print\\(\"([^\"]*)\"\\); (This tells The WheeC Engine what is the Whee Code that refers to this function)

Replacement: printf(\"%s\\n\"); (This tells The WheeC Engine what is the C code to put in place of this function)

The famous "Hello World!"

You can simply run a "Hello world!" program in Whee by writing:

=WHEE= print("Hello world!"); It should show "Hello world!":
