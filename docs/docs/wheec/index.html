WheeC and The WheeC Engine

Understanding WheeC

WheeC is a process of turning Whee code to C code.

Then, the C code gets executed.

This will allow users to easily modify the code while being blazingly fast.

The WheeC Engine

A WheeC Engine will do what WheeC needs to do: Turn Whee to C.

The default WheeC Engine that comes with Whee is whee-to-c.

You can try it any time by running:

whee-to-c [script].whee

Which will output the C code with the top line being "=== Generated C Code ===" to ensure that it is a valid WheeC Engine.

You can try modifying it's code by putting a file wheeinit/required.c

You can also try adding wheeinit/optional.c if isn't essential

Modules will use wheeinit/modules/[module]/[optional/required].c
